Progressive Enhancement Writing Assignment

Progressive enhancement is from the reading provided, a “content up” way to view the design process for creating web related content, it starts with the raw content, then adds layers upon it. When compared to the “graceful degradation” philosophy, progressive enhancement is a more sustainable way to produce content, relying not on the browser features, but upon underlying content itself, and building upon it with the design, and then topping it off with functionality, all without relying upon the newest browser specs to perform well. This leaves aging machines with old ways of accessing the web more availability to use this content. And frankly, based on how old some of the tech in the USA is, this is for the best.
